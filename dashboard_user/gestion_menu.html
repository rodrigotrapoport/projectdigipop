Panel de Control - Configuración del Menu y Submenu

Si es tu primera vez aquí vas a encontrar todo lo que es relacionado con la configuración del menu y submenus. En el caso de que tengas una categoría de productos o servicios que tenga su propio submenu anclado a un menu "padre", puedes utilizar este mecanismo. En el caso de que no sea necesario cambiar las jerarquias el sistema lo hace de forma automática.

En el caso de que quieras crear una menu o submenu nuevo debes realizar las siguientes acciones:

Primero crear el menu o crear el submenu relacionando con la página de la categoría de productos o el servicio. Recuerda que si vas a realizar un menu nuevo, debes poner en la jerarquia "HERMANO". Si quieres crear un submenu, debes poner y refenrenciar cual es el "PADRE".

Si necesitas ubicar de formas distintas a las que has realizado puedes utilizar la tabla.

Recuerda que sino entiendes algo podes consultar los tutoriales de configuración o pedir asistencia técnica a nuestro equipo a través de Messenger. Te recomendamos para que hagas la configuración completa y de forma exitosa.
