Panel de Control - Configuración de Carousel / Slides

Si es tu primera vez aquí vas a encontrar todo lo que es relacionado con la configuración de los sliders que se muestran en la Página principal, Categorías de Productos o Servicios. Existen dos tipos de slides: 1) sin formularios (ideal para e-commerce) y 2) con formulario integrado (ideal para venta de servicios). Los slides pueden tener títulos y subtitulos, como también un boton que llame a la acción.

Para que un slide se muestre en un conjunto de slides debes crear una categoría de slides para que después puedas relacionarlo con un servicio o categoría de productos

Recuerda que sino entiendes algo podes consultar los tutoriales de configuración o pedir asistencia técnica a nuestro equipo a través de Messenger. Te recomendamos para que hagas la configuración completa y de forma exitosa.
